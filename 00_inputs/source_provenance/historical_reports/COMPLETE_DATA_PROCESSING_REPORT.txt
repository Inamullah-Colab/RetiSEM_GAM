5.1 MISSING DATA THRESHOLD RULE
   
   Rule applied: Remove variables with >30% missing data
   
   Rationale:
   - Variables with >30% missing have limited information
   - Difficult to impute reliably with high missingness
   - May introduce bias in causal inference
   - Keep only interpretable variables
   
   This is a standard approach in epidemiology:
   ✓ More rigorous than keeping all variables
   ✓ Ensures data quality for causal inference
   ✓ Reduces noise in downstream analysis

SECTION 6: MISSING DATA IMPUTATION
================================================================================

6.1 IMPUTATION METHOD & RATIONALE

   Method: Median imputation for numeric variables
   
   Why median imputation?
   ✓ Robust to outliers (unlike mean)
   ✓ Preserves distribution shape
   ✓ Simple and reproducible
   ✓ Commonly used in epidemiological research
   ✓ Suitable for ≤30% missing data (used conservative estimate)
   
   NOTE: For very robust imputation with highly missing data, 
         MICE (Multivariate Imputation by Chained Equations) would be preferred,
         but median is appropriate for this threshold.
