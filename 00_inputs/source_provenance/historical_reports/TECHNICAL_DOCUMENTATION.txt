SECTION 1: READING XPT FILES
================================================================================

File Format: SAS XPORT (XPT)
Data Source: NHANES website (CDC/NCHS)
Files: 9 XPT files in same directory

Problem: Many libraries fail on XPT files
Solution: Use pandas.read_sas(format='xport')

Code:

```python
import pandas as pd
from pathlib import Path

data_dir = Path(r"path/to/NHANES_Dataset")

# Read XPT file
file_path = data_dir / "DEMO_D.xpt"
df = pd.read_sas(str(file_path), format='xport')

# Check results
print(df.shape)           # (10348, 43)
print(df.columns[:5])    # First 5 columns
print(df.head())         # Preview
```

Result:
  Successfully loads all 9 files
  DEMO_D.xpt → 10,348 rows × 43 cols
  BMX_D.xpt → 9,950 rows × 27 cols
  ... (and 7 others)

Key Variable: SEQN (Participant ID) present in all files

================================================================================
SECTION 2: MERGING DATASETS
================================================================================

Strategy: LEFT JOIN on SEQN (Participant ID)
Base: DEMO_D (largest dataset, 10,348 rows)
Order: DEMO_D → BMX_D → BPX_D → HDL_D → LEXABPI → SLQ_D → SMQ_D → TCHOL_D → TRIGLY_D

Code:

```python
import pandas as pd

# Load all datasets
demo = pd.read_sas("DEMO_D.xpt", format='xport')
bmx = pd.read_sas("BMX_D.xpt", format='xport')
bpx = pd.read_sas("BPX_D.xpt", format='xport')
hdl = pd.read_sas("HDL_D.xpt", format='xport')
lex = pd.read_sas("LEXABPI.xpt", format='xport')
slq = pd.read_sas("SLQ_D.xpt", format='xport')
smq = pd.read_sas("SMQ_D.xpt", format='xport')
tchol = pd.read_sas("TCHOL_D.xpt", format='xport')
trigly = pd.read_sas("TRIGLY_D.xpt", format='xport')

# Merge on SEQN
merged = demo.copy()
merged = merged.merge(bmx, on='SEQN', how='left')
merged = merged.merge(bpx, on='SEQN', how='left')
merged = merged.merge(hdl, on='SEQN', how='left')
merged = merged.merge(lex, on='SEQN', how='left')
merged = merged.merge(slq, on='SEQN', how='left')
merged = merged.merge(smq, on='SEQN', how='left')
merged = merged.merge(tchol, on='SEQN', how='left')
merged = merged.merge(trigly, on='SEQN', how='left')

# Save
merged.to_csv("NHANES_merged.csv", index=False)
print(f"Merged: {merged.shape}")  # (10348, 187)
```

Result: NHANES_merged.csv (10,348 × 187)

Verification:
  - No duplicate SEQN values
  - All rows preserved (left join kept all from DEMO_D)
  - Column count: 43 + (27+28+3+16+28+39+3+8) = 187 ✓

SECTION 3: COMPLETE CASE FILTERING
================================================================================

Goal: Keep only participants with complete data on key variables
Key Variables: RIDAGEYR, RIAGENDR, BMXBMI, BPXSY1, BPXDI1, BPXPLS

Code:

```python
import pandas as pd
import numpy as np

# Load merged data
df = pd.read_csv("NHANES_merged.csv")
print(f"Initial: {df.shape}")  # (10348, 187)

# Define key variables
key_vars = ['RIDAGEYR', 'RIAGENDR', 'BMXBMI', 'BPXSY1', 'BPXDI1', 'BPXPLS']

# Check availability
for var in key_vars:
    n_missing = df[var].isnull().sum()
    pct = (n_missing / len(df)) * 100
    print(f"{var}: {len(df) - n_missing} / {len(df)} ({100-pct:.1f}%)")

# RESULTS:
# RIDAGEYR: 10348 / 10348 (100.0%)
# RIAGENDR: 10348 / 10348 (100.0%)
# BMXBMI:   8949 / 10348 (86.5%)
# BPXSY1:   6668 / 10348 (64.4%)   ← BOTTLENECK
# BPXDI1:   6668 / 10348 (64.4%)   ← BOTTLENECK
# BPXPLS:   7441 / 10348 (71.9%)

# Filter: Keep only complete cases on ALL key variables
complete_mask = df[key_vars].notna().all(axis=1)
df_clean = df[complete_mask]

print(f"After filtering: {df_clean.shape}")  # (6606, 187)
print(f"Retention: {(len(df_clean)/len(df))*100:.1f}%")  # 63.8%

# Save
df_clean.to_csv("NHANES_cleaned_causal.csv", index=False)
```

Result: NHANES_cleaned_causal.csv (6,606 × 187)

Key Insight: Blood pressure measurements only on 64.4% = bottleneck

6.2 IMPUTE NUMERIC COLUMNS (MEDIAN)

```python
import pandas as pd
import numpy as np

df = pd.read_csv("NHANES_filtered_causal.csv")

# Get numeric columns
numeric_cols = df.select_dtypes(include=[np.number]).columns.tolist()

# Impute with median
for col in numeric_cols:
    if df[col].isnull().any():
        median_val = df[col].median()
        df[col].fillna(median_val, inplace=True)
        print(f"Imputed {col}: median = {median_val}")

# Check results
print(f"Missing after numeric imputation: {df.isnull().sum().sum()}")
```

6.3 IMPUTE NON-NUMERIC COLUMNS (MODE)

```python
import pandas as pd
import numpy as np

df = pd.read_csv("NHANES_filtered_causal.csv")

# Get non-numeric columns with missing
non_numeric_cols = df.select_dtypes(exclude=[np.number]).columns.tolist()

for col in non_numeric_cols:
    if df[col].isnull().any():
        mode_val = df[col].mode()[0] if len(df[col].mode()) > 0 else 'Unknown'
        df[col].fillna(mode_val, inplace=True)
        print(f"Imputed {col}: mode = {mode_val}")

# Final check
total_missing = df.isnull().sum().sum()
print(f"Total missing after all imputation: {total_missing}")  # Should be 0

# Save
```

SECTION 8: REPRODUCIBILITY CHECKLIST
================================================================================

To reproduce this entire analysis:

□ Download 9 XPT files from NHANES
  Website: https://wwwn.cdc.gov/nchs/nhanes/
  
□ Place all XPT files in single directory

□ Run in Python (in that directory):

```python
# Step 1: Read and merge
import pandas as pd
files = ['DEMO_D', 'BMX_D', 'BPX_D', 'HDL_D', 'LEXABPI', 'SLQ_D', 'SMQ_D', 'TCHOL_D', 'TRIGLY_D']
dfs = {f: pd.read_sas(f"{f}.xpt", format='xport') for f in files}
merged = dfs['DEMO_D']
for f in files[1:]:
    merged = merged.merge(dfs[f], on='SEQN', how='left')
merged.to_csv("NHANES_merged.csv", index=False)
print(f"Merged: {merged.shape}")  # Should be (10348, 187)

# Step 2: Complete cases
key_vars = ['RIDAGEYR', 'RIAGENDR', 'BMXBMI', 'BPXSY1', 'BPXDI1', 'BPXPLS']
df_clean = merged[merged[key_vars].notna().all(axis=1)]
df_clean.to_csv("NHANES_cleaned_causal.csv", index=False)
print(f"Complete cases: {df_clean.shape}")  # Should be (6606, 187)

# Step 3: Remove >30% missing
missing_pcts = (df_clean.isnull().sum() / len(df_clean) * 100)
vars_keep = missing_pcts[missing_pcts <= 30].index.tolist()
df_reduced = df_clean[vars_keep]
df_reduced.to_csv("NHANES_filtered_causal.csv", index=False)
print(f"Filtered: {df_reduced.shape}")  # Should be (6606, 100)

# Step 4: Impute missing
import numpy as np
numeric_cols = df_reduced.select_dtypes(include=[np.number]).columns
for col in numeric_cols:
    df_reduced[col].fillna(df_reduced[col].median(), inplace=True)
non_numeric_cols = df_reduced.select_dtypes(exclude=[np.number]).columns
for col in non_numeric_cols:
    if df_reduced[col].isnull().any():
        df_reduced[col].fillna(df_reduced[col].mode()[0], inplace=True)
df_reduced.to_csv("NHANES_final_causal.csv", index=False)
print(f"Imputed: {df_reduced.shape}")  # Should be (6606, 100)
print(f"Missing: {df_reduced.isnull().sum().sum()}")  # Should be 0
```

Expected outputs:
  ✓ Merged: (10348, 187)
  ✓ Complete cases: (6606, 187)
  ✓ Filtered: (6606, 100)
  ✓ Imputed: (6606, 100)
  ✓ Missing: 0

================================================================================
TROUBLESHOOTING COMMON ISSUES
================================================================================

Issue 1: XPT file won't read
  Error: "Invalid file, or file has unsupported features"
  Fix: Use pd.read_sas(format='xport') from statsmodels/scipy
       NOT pd.read_sas() alone
       
Issue 2: SEQN not found in one dataset
  Error: KeyError: 'SEQN'
  Fix: Check file is correct XPT format
       Some NHANES files use different ID variable names

Issue 3: Merge creates too many columns/duplicates
  Problem: Duplicate column names after merge
  Fix: Use merge(on='SEQN', how='left', suffixes=('', '_B'))
       to add suffixes to avoid name collisions

Issue 4: Still has missing values after imputation
  Problem: Non-numeric columns not imputed
  Fix: Use np.select_dtypes(exclude=[np.number]) 
       to identify categorical columns
       
Issue 5: Outliers look impossible
  Example: BP = 270, Pulse = 340
  Action: Check NHANES codebook for valid ranges
          Consider excluding in sensitivity analysis
