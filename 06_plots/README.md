## Plot conventions

- `TE` is shown in blue
- `NDE` is shown in green
- `NIE` is shown in red
- NIE-significant pathways are highlighted in orange
- `*` marks NIE-significant pathways
- `| Z,G` is retained to show covariate adjustment
